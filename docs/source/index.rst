Welcome to the Viking's documentation!
======================================

.. note::

   This project is under active development.

**Viking** is a large, Linux compute cluster with many nodes, CPUs, GPUs, lots of storage and memory. It is housed in Sweden in the fabulous `EcoDataCentre <https://ecodatacenter.tech/>`_, which is one of the most sustainable data centres in the whole world!
